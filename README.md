


# AI Meeting Assistant
A simple AI-powered meeting management application that helps users upload meeting recordings and automatically generate transcripts, summaries, action items, decisions, and follow-up tasks.

## Features
* User Signup and Login
* Create and Delete Meetings
* Upload Meeting Audio
* Generate Meeting Transcript using AI
* Generate Meeting Summary
* Extract Action Items
* Extract Decisions Taken During Meetings
* Generate Follow-Up Tasks
* User Profile Page
* Clean and Responsive UI

## Tech Stack

### Frontend

* React.js
* React Router DOM
* Tailwind CSS
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

### AI Services

* AssemblyAI (Speech-to-Text)
* Groq API (AI Processing)

## How It Works

1. User creates an account and logs in.
2. User creates a new meeting.
3. User uploads the meeting audio file.
4. AI converts the audio into text (transcript).
5. AI generates:

   * Summary
   * Action Items
   * Decisions
   * Follow-Ups
6. User can view all meeting details in one place.
